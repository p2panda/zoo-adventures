# beep-boop

[p2panda](https://github.com/p2panda/handbook) React component to play the Zoo Adventure game.

## Development

```bash
# Install NodeJS dependencies
npm install

# Check linter errors
npm run lint

# Build library and demo
npm run build

# Watch for changes and automatically run `build`
npm run watch

# Start server to host demo (needs building first)
npm run serve
```
